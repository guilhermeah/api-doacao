# API REST - Sistema de Doação

API REST desenvolvida para o Trabalho de Conclusão de Curso (TCC), com o objetivo de gerenciar doadores, ONGs, doações e demais recursos de uma plataforma de apoio social.

---

## Sobre o projeto

O projeto tem como finalidade conectar doadores a instituições e beneficiários, facilitando o processo de registro, acompanhamento e organização das doações.

A proposta foi pensada para funcionar em dois contextos:

- **Painel desktop/web da ONG**: gerenciamento administrativo
- **Aplicativo mobile do doador**: registro e acompanhamento de doações

Esta API representa o backend central do sistema, responsável por disponibilizar os dados e regras de negócio para ambas as interfaces.

---

## Objetivo

Desenvolver uma solução tecnológica para auxiliar no controle e na distribuição de doações, promovendo organização, transparência e acessibilidade no processo.

---

## Tecnologias utilizadas

- **Java 17**
- **Spring Boot**
- **Spring Data JPA**
- **PostgreSQL**
- **Maven**
- **Swagger / OpenAPI**
- **Git e GitHub**
